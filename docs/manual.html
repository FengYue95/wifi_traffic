路段拥堵指数

接口描述

该接口用于返回某条路段拥堵指数的数据。通过URL请求，传入路段的编号，返回该路段的拥堵指数

URL

http://ip地址:8800/wifitraffic/congestion_index_by_uri/(segmentid)

参数说明

参数 必需 类型 说明
segmentid true int 路段编号

Method

GET(直接在URL中传入参数

返回数据格式

JSON

返回数据说明

参数 类型 说明
segmentid double 路段拥堵指数
time string 时间

{"3": 4, "time": "20161031005000"}
